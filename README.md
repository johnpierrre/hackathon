# Structure and deployment

## Structure and languages

Memifier:
    -cmd directory for the backend (written in GO)
        -api directory for api and handler relateds functions (GO)
        
    -public directory for the frontend (written in react.js)
## Deployment

Frontend:
    - "npm install"
    - "npm start"
Backend:
    - "cd cmd"
    - "go run main.go"
